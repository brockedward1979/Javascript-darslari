# Shartli Operatorlar:

```javascript
if (shart1) {
	// kod
}
else if (shart2) {
	// kod
}
else if (shart3) {
	// kod
} else {
	//
}
```

Ushbu proyekt `if` operatorining ikki xil ishlash usulini tushuntiradi.
